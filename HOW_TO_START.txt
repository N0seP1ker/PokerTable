┌──────────────────────────────────────────────────────────────┐
│  EASIEST METHOD - ONE DOUBLE-CLICK                           │
└──────────────────────────────────────────────────────────────┘

  Windows Users:
  ──────────────
  1. Find "start.bat" in this folder
  2. Double-click it
  3. Wait for browsers to open
  4. Done! 🎉

  Mac/Linux Users:
  ────────────────
  1. Open Terminal in this folder
  2. Type: npm start
  3. Press Enter
  4. Done! 🎉


┌──────────────────────────────────────────────────────────────┐
│  WHAT HAPPENS WHEN YOU START                                 │
└──────────────────────────────────────────────────────────────┘

  ✓ Two new windows will open:
    - Backend Server (Port 3001)
    - Frontend Server (Port 3000)

  ✓ After ~8 seconds, your browser opens automatically to:

┌──────────────────────────────────────────────────────────────┐
│  ALTERNATIVE: USING COMMAND LINE                             │
└──────────────────────────────────────────────────────────────┘

  Method 1 - Simple (runs both servers):
  ──────────────────────────────────────
  npm run dev

  Method 2 - Manual (more control):
  ─────────────────────────────────
  Terminal 1:  cd backend && npm run dev
  Terminal 2:  cd frontend && npm run dev


┌──────────────────────────────────────────────────────────────┐
│  HOW TO PLAY                                                 │
└──────────────────────────────────────────────────────────────┘

  1. Open http://localhost:3000 in your browser
  2. Create a new room (or join existing)
  3. Click on an empty seat (marked "SIT")
  4. Share the room link with friends
  5. Chat with other players (bottom-left)
  6. Host clicks "Start Game" when ready


┌──────────────────────────────────────────────────────────────┐
│  STOPPING THE SERVERS                                        │
└──────────────────────────────────────────────────────────────┘

  ✓ Close the two server windows that appeared
  OR
  ✓ Press Ctrl+C in the terminal


┌──────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                             │
└──────────────────────────────────────────────────────────────┘

  Problem: "Port already in use"
  Solution: Stop other apps using ports 3000 or 3001

  Problem: "Cannot find module"
  Solution: Run these commands:
    cd backend && npm install
    cd ../frontend && npm install

  Problem: Scripts won't run
  Solution: Make sure Node.js is installed:
    node --version  (should show v16 or higher)
